# Ana-Sofia-Portfolio

## Overview
This is my personal portfolio website. The website features a modern, interactive layout with smooth scrolling, animated card elements, and a responsive design suitable for desktop, tablet, and mobile devices.

## Features
- **Project Showcase**: Displays projects fetched from an API with lazy loading and horizontal scrolling.
- **Filter by Scope**: Users can filter projects by their prefered scope using a dropdown menu.
- **Custom Cards**: Each project is displayed using a custom `<new-card>` web component with image, title, tags, description, and external links.
- **Animated Reveal**: Elements appear with smooth reveal animations as the user scrolls.
- **Smooth Scroll Buttons**: Navigation buttons for scrolling vertically between sections and horizontally through project cards.
- **Contact Links**: Quick access to email, Behance, and GitHub profiles.

## Structure
- **index.html**: Main HTML structure with sections for work and contact information.
- **js/fetch.js**: Handles fetching projects from the API and dynamically rendering cards with lazy loading.
- **js/card.js**: Defines the `<new-card>` custom element for displaying project cards.
- **js/buttonscroll.js**: Handles scroll buttons and reveal animations.
- **js/menuhamburguer.js**: Handles mobile hamburger menu interactions.
- **styles.css**: Global styles for the website.
- **card.css**: Styles specifically for the project cards.
